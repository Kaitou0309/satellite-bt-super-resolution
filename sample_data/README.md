# Public Example Data

`amsr2_example.h5` is one compressed, full-size LR/HR pair for software validation:

```text
L/bt    (96, 100), float32, Kelvin
H/bt    (384, 400), float32, Kelvin
```

It can be used for inference, metrics, plots, and schema tests. It is not a representative training dataset.

Original data for this value added data product was provided by Japan Aerospace Exploration Agency. See `docs/DATA.md` for construction details and attribution requirements.
